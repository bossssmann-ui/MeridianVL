# HANDOFF — MeridianVL

Технический документ для старта разработки. Читать вместе с README.md (концепция) и ROADMAP.md (порядок работ).

## 1. Ключевые решения

| Вопрос | Решение |
|---|---|
| Стек | Статика: HTML5 + CSS3 + Vanilla JS (ES6+), без сборщиков |
| Данные | JSON-файлы в `/data`; **контент пре-рендерится в HTML build-скриптом** (см. SEO.md, раздел 0), JS — только фильтры и интерактив |
| URL | Каждый маршрут и авто — отдельный HTML с ЧПУ: `/marshruty/ostrov-russkiy/`, `/avto/toyota-rav4/` |
| Бронирование | Заявка: форма + WhatsApp / Telegram / MAX + телефон. Онлайн-оплаты нет |
| Дизайн | Яркий современный, mobile-first. НЕ копировать layout primecar-rent.ru |
| Карты | Яндекс.Карты JS API (маршруты, запретные зоны, офис) |
| Фото | Временно — плейсхолдеры/стоки; заказчик заменит своими в процессе |

## 2. Дизайн-система

```css
:root {
  --c-primary: #0B3C5D;   /* глубокий морской синий */
  --c-accent:  #FF6B35;   /* коралловый — CTA, акценты */
  --c-teal:    #17A2B8;   /* вторичный акцент */
  --c-bg:      #F7F9FC;   /* фон */
  --c-surface: #FFFFFF;
  --c-text:    #16232E;
  --c-muted:   #5E7183;
  --radius: 16px;
  --shadow: 0 8px 24px rgba(11,60,93,.10);
}
```

- Шрифты: заголовки — Unbounded (Google Fonts), текст — Manrope. Без КАПСА.
- Брейкпоинты: 480 / 768 / 1024 / 1280.
- Иконки: Lucide (SVG-спрайт), не эмодзи.
- Анимации: только transform/opacity, короткие (150–250 мс); появление секций через IntersectionObserver.

Жёсткие анти-паттерны (чтобы не повторить конкурента): никаких табов классов как единственной навигации по каталогу; форма заявки не в hero первым экраном; карточки авто не в стиле «капс + 4 колонки характеристик».

## 3. UX-ориентиры по страницам

**Главная:** hero (заголовок + выбор дат + CTA «Подобрать авто») → «Куда поехать в эти выходные» (карточки маршрутов) → популярные авто (карусель) → преимущества → как это работает (таймлайн 4 шага) → отзывы (виджет Яндекс) → FAQ (5 популярных, ссылка на все) → контакты. На мобильном — нижний таб-бар: Автопарк / Маршруты / Позвонить.

**Автопарк (catalog.html):** слева/сверху фильтры (класс, цена-слайдер, привод, топливо, число мест, коробка), сортировка (цена ↑↓, год), поиск по модели. Карточка: фото, модель, год, 3–4 иконки-характеристики, «от N ₽/сутки», кнопка «Забронировать» → модал заявки с подстановкой модели.

**Маршруты (routes.html + route.html):** список с фильтрами (тип: с детьми / романтика / фото / гастро / офроуд; длительность: полдня / день / выходные; дорога: асфальт / грунт / 4WD). Страница маршрута: обложка, легенда, карта с точками, факты (км, время, сезон), точки маршрута с советами, блок «Авто под этот маршрут» (2–3 рекомендации из cars.json по тегам) с расчётом стоимости, CTA. Отдельно — карта запретных зон Приморья с пояснением.

**Условия (terms.html):** таблица-сравнение классов (возраст / стаж / залог / лимит 200 км/сутки / перепробег), документы, порядок оформления.

## 4. Схемы данных

`data/cars.json`:
```json
{
  "id": "toyota-rav4-2021",
  "brand": "Toyota", "model": "RAV4", "year": 2021,
  "class": "crossover",            // econom|comfort|premium|minivan|crossover
  "pricePerDay": 6500,
  "discounts": [{"fromDays": 3, "pct": 5}, {"fromDays": 7, "pct": 10}],
  "engine": 2.0, "fuel": "petrol", // petrol|hybrid|electric
  "drive": "4wd",                  // fwd|rwd|4wd
  "transmission": "at", "seats": 5,
  "deposit": 15000, "minAge": 25, "minExperience": 5,
  "tags": ["offroad", "family", "long-trip"],
  "photos": ["img/cars/rav4-1.jpg"],
  "available": true
}
```

`data/routes.json`:
```json
{
  "id": "russky-island-day",
  "title": "Остров Русский без толп",
  "teaser": "Кампус ДВФУ, Новосильцевская батарея и дикие бухты за один день",
  "type": ["photo", "family"],
  "duration": "day",               // half-day|day|weekend
  "distanceKm": 90, "driveTime": "2,5 ч за рулём",
  "season": "май–октябрь",
  "road": "asphalt",               // asphalt|gravel|4wd-only
  "points": [
    {"name": "Мост на Русский", "coords": [43.06, 131.90],
     "tip": "Смотровая на Орлиной сопке до моста — лучший кадр утром"}
  ],
  "recommendedTags": ["family"],   // матчинг с cars.json.tags
  "recommendedClass": ["comfort", "crossover"],
  "warnings": "Не съезжать на грунтовки в глубь острова — запретная зона",
  "photos": ["img/routes/russky-1.jpg"]
}
```

Связка маршрут→авто: пересечение `route.recommendedTags` с `car.tags`, фолбэк по `recommendedClass`. Стоимость под маршрут: `pricePerDay × дни − скидка`, показать лимит 200 км/сутки и перепробег, если `distanceKm` близко к лимиту.

## 5. Формы и интеграции

- Поля заявки: имя, телефон, даты (с/по), авто/пожелания, способ связи (WhatsApp / Telegram / MAX / звонок), чекбокс согласия ПДн (обязателен, ссылки на политику и согласие).
- Отправка: POST на лёгкий эндпоинт → Telegram-бот владельцу (или Formspree/SMTP на старте). Без бэкенда с БД.
- Быстрые ссылки: `https://wa.me/<номер>?text=<модель, даты>` — текст генерируется из контекста карточки.
- Яндекс.Метрика + цели на отправку форм и клики по мессенджерам.
- Контакты, телефоны, адрес, реквизиты — **плейсхолдерами** `[PHONE]`, `[ADDRESS]`, `[INN]` до предоставления заказчиком.

## 6. SEO и юридическое

- Каждая страница: уникальные title/description, h1; Open Graph.
- Schema.org: LocalBusiness (главная), Product/Offer (карточки авто), FAQPage (faq), Article (маршруты).
- sitemap.xml, robots.txt, ЧПУ для маршрутов (route.html?id=… на старте, позже — отдельные html).
- Обязательно: политика обработки ПДн, согласие, cookie-баннер (152-ФЗ). Дисклеймер «не является публичной офертой».
- Тексты не копировать у конкурента — вся структура и формулировки пишутся с нуля.

## 7. Definition of Done (запуск)

- Lighthouse mobile: Performance ≥ 90, SEO ≥ 95, Accessibility ≥ 90.
- Все страницы адаптивны 360px–1440px, проверены на реальном телефоне.
- Формы доставляют заявку владельцу и показывают подтверждение.
- В каталоге ≥ 15 авто, маршрутов ≥ 6, FAQ ≥ 12.
- Никаких заимствованных текстов/фото с primecar-rent.ru.
