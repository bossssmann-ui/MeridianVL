# MeridianVL — прокат автомобилей во Владивостоке

Современный сайт проката автомобилей. Яркий, динамичный, «сайт-приложение» — полная противоположность классическим лендингам конкурентов по подаче, при равном или большем наполнении.

**Стек:** статика — HTML5 / CSS3 / Vanilla JS. Без сборщиков и фреймворков на старте. Данные автопарка и мест — в JSON, рендер на клиенте.

---

## 1. Анализ конкурента (primecar-rent.ru)

Что у него есть и работает:

- Одностраничный светлый лендинг: hero → преимущества (200+ авто, 50+ моделей, рейтинг 5.0, подача в аэропорт) → каталог с табами классов → условия аренды → «как происходит оформление» (4 шага) → отзывы (Яндекс Карты) → FAQ (9 вопросов) → контакты.
- Каталог ~60 авто в 5 категориях (Эконом, Комфорт, Премиум, Минивэны, Кроссоверы), карточка: фото, цена «от X руб/сутки», год, объём, привод, топливо.
- Условия по классам: возраст/стаж/залог (Эконом: 20 лет / 1,5 года / от 5 000 ₽; Комфорт: 21 / 3 / от 10 000 ₽; Премиум: 25 / 5 / от 15 000 ₽).
- Подробный FAQ: бронирование (предоплата 3 000 ₽), лимит пробега 200 км/сутки, запретные зоны Приморья, ДТП, поломки, страховка, ответственность.
- Заявка через форму / WhatsApp / Telegram / MAX / звонок. Онлайн-оплаты нет.
- Фишка: раздел «Топ мест во Владивостоке» — подборка достопримечательностей.

Слабые места, которые мы используем:

- Дизайн шаблонный (конструктор), нет мобильного «вау», карточки перегружены капсом.
- «Топ мест» — просто список мест, не связан с автопарком и не помогает выбрать машину.
- Нет фильтров каталога (только табы), нет калькулятора стоимости, нет привязки маршрутов к запретным зонам.

## 2. Наша концепция

Всё то же по наполнению (не меньше), но:

- **Структура — многостраничная**: Главная / Автопарк / Маршруты / Условия / FAQ / Контакты. У конкурента — один лендинг; у нас каждая страница работает на SEO отдельно.
- **Подача** — яркий современный UI: смелая палитра, крупные карточки, живые фильтры, микроанимации, mobile-first.
- **Каталог сильнее**: фильтры (класс, цена, привод, топливо, коробка, число мест) + сортировка + мгновенный поиск, а не просто табы.
- **Калькулятор стоимости**: даты + класс → примерная сумма с учётом скидок за длительность, сразу на карточке.
- **Заявка**: форма + WhatsApp / Telegram / MAX / звонок (как у конкурента, паритет). Онлайн-оплата — в бэклоге.

## 3. Фишка: «Маршруты Приморья» (переработка «Топ мест»)

У конкурента — статичный список достопримечательностей. Мы разворачиваем идею от «куда сходить» к «куда поехать на нашей машине» — фишка продаёт аренду, а не туризм.

**Формат: не места, а готовые маршруты выходного дня.** Каждый маршрут — карточка-сценарий:

- Название и легенда («Маяки и мосты: классика Владивостока за один день», «Остров Русский без толп», «Гастротур: от Миллионки до крабов на Шаморе», «Тавричанка и Хасанский рассвет»).
- Карта с точками (Яндекс.Карты API), километраж, время в пути, лучший сезон.
- Сложность дороги: асфальт / грунтовка / только 4WD — и сразу **рекомендованные авто из нашего парка** с ценой и кнопкой «Забронировать под маршрут». Это ключевая связка: маршрут → машина → заявка.
- Практика: где заправиться, где поесть, парковка, лучшая точка для фото, во сколько выезжать.
- **Проверка на запретные зоны**: у конкурента ограничения спрятаны в FAQ; у нас каждый маршрут заранее «легален», а отдельная страница с картой запретных зон Приморья — сама по себе полезность, которой нет ни у кого.

Дополнительные механики (по мере развития):

- Фильтр маршрутов: с детьми / романтика / фототур / рыбалка / гастро / офроуд; по длительности (полдня / день / выходные).
- «Маршрут + авто» пакетом: примерная стоимость аренды под конкретный маршрут считается автоматически (дни × тариф + перепробег, лимит 200 км/сутки учтён).
- Сезонный блок на главной: «Куда поехать в эти выходные» — 2–3 актуальных маршрута (лёд на Русском зимой, маяк Токаревского на закате летом).
- UGC-петля: гости присылают фото с маршрутов → галерея «Проехали с MeridianVL» (соцдоказательство + бесплатный контент).
- Каждый маршрут — отдельная SEO-страница («что посмотреть в Приморье на машине», «куда съездить из Владивостока на выходные» — трафик, который конкурент не собирает).

Отличие от конкурента по сути: у них контент-виджет «Топ мест», у нас — **инструмент планирования поездки, встроенный в воронку аренды**.

## 4. Дизайн-направление (в корне другое)

- Палитра: глубокий морской синий + коралловый/оранжевый акцент, светлые подложки. Конкурент — нейтральный белый корпоратив; мы — энергия и движение.
- Типографика: крупные жирные заголовки (Unbounded / Manrope), без капса в карточках.
- Hero: не статичный баннер с формой, а полноэкранный блок с поиском по датам и «живыми» карточками маршрутов.
- Карточка авто: горизонтальная на десктопе, свайп-карусель на мобильном; характеристики иконками, цена крупно, кнопка одна.
- Навигация: липкий верхний бар + плавающая кнопка WhatsApp; на мобильном — нижний таб-бар (Автопарк / Маршруты / Позвонить).
- Расположение блоков главной намеренно другое: hero-поиск → маршруты недели → автопарк (фильтры) → калькулятор → как это работает → отзывы → FAQ → контакты.

```
│   └── faq.json

```
