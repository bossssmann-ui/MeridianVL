# ROADMAP — MeridianVL

Порядок работ по созданию сайта. Каждый этап заканчивается рабочим результатом, который можно посмотреть в браузере.

## Этап 0 — Фундамент

- [ ] Создать репозиторий, залить README / ROADMAP / HANDOFF
- [ ] Каркас проекта: структура папок, базовый HTML-шаблон (шапка, подвал, мета-теги)
- [ ] Build-скрипт пре-рендера: JSON → готовые HTML-страницы с ЧПУ (требование SEO.md, раздел 0)
- [ ] Собрать семантическое ядро по Вордстату (регион Владивосток) — до написания текстов (SEO.md, раздел 4)
- [ ] Дизайн-токены в CSS: палитра, типографика, отступы, брейкпоинты
- [ ] Подключить шрифты, иконки, favicon

## Этап 1 — Данные

- [ ] `data/cars.json` — схема карточки авто + первые 10–15 машин (остальные добиваем по мере готовности фото)
- [ ] `data/routes.json` — схема маршрута + 3 первых маршрута (тексты, точки, рекомендованные авто)
- [ ] `data/places.json`, `data/food.json`, `data/events.json` — наполнить из PLACES.md, сверить актуальность
- [ ] `data/faq.json` — 12+ вопросов по группам (бронирование, условия, в поездке, ДТП/страховка)
- [ ] Свод условий аренды по классам (возраст, стаж, залог, лимит пробега)

## Этап 2 — Главная страница

- [ ] Hero с поиском по датам и CTA
- [ ] Блок «Куда поехать в эти выходные» (2–3 маршрута)
- [ ] Превью автопарка (популярные модели) + переход в каталог
- [ ] Преимущества, «как это работает» (таймлайн), отзывы (Яндекс), FAQ-превью, контакты
- [ ] Мобильная версия: нижний таб-бар, плавающая кнопка WhatsApp

## Этап 3 — Автопарк

- [ ] Каталог из cars.json: карточки, фильтры (класс, цена, привод, топливо, места), сортировка, поиск
- [ ] Калькулятор стоимости (даты → сумма, скидки за длительность)
- [ ] Заявка на авто: форма + ссылки WhatsApp / Telegram / MAX с подстановкой модели

## Этап 4 — Маршруты Приморья (фишка)

- [ ] Страница-список маршрутов с фильтрами (тип отдыха, длительность, сложность дороги)
- [ ] Шаблон страницы маршрута: карта (Яндекс.Карты API), точки, километраж, сезон, практические советы
- [ ] Связка «маршрут → рекомендованные авто → заявка» с расчётом стоимости под маршрут
- [ ] Страница/слой карты запретных зон Приморья
- [ ] Довести число маршрутов до 6–8

## Этап 5 — Контент и вспомогательные страницы

- [ ] Условия аренды (таблица-сравнение классов), FAQ, Контакты, Юрлицам
- [ ] Политика ПДн, согласие, cookie-баннер
- [ ] Замена временных фото на свои (авто и маршруты) — идёт параллельно всем этапам

## Этап 6 — Качество и запуск

- [ ] Адаптив на реальных устройствах, Lighthouse ≥ 90 (Performance / SEO / A11y)
- [ ] SEO-чеклист из SEO.md полностью: Schema.org (AutoRental, Product, FAQPage, TouristTrip, Event), sitemap.xml, robots.txt с разрешением ИИ-ботов, llms.txt
- [ ] Проверка контента с отключённым JS — всё видно (SEO.md, раздел 0)
- [ ] Подключение отправки форм (Telegram-бот или почта), Яндекс.Метрика с целями
- [ ] Домен, хостинг с точкой присутствия в РФ, HTTPS, публикация
- [ ] Яндекс.Вебмастер (регион «Владивосток», sitemap), Google Search Console
- [ ] Регистрация в Яндекс Бизнес / 2ГИС / Google Business, единый NAP, проверка отзывов

- Партнёрская программа: шорт-лист организаций из маршрутов, готовые предложения, промокоды, бейджи «Бонус для наших гостей» (см. PLACES.md, раздел 6)
- Календарь занятости авто и онлайн-предоплата брони
- UGC-галерея «Проехали с MeridianVL»
- Сезонная ротация маршрутов, блог для SEO
- Личный кабинет / повторные брони
- Английская версия (туристы из Азии)
- Дзен-канал с маршрутами, инфоповоды для местных СМИ (SEO.md, раздел 7)
- Ежеквартальная проверка видимости в ИИ-ответах: Алиса, ChatGPT, Perplexity (SEO.md, раздел 8)
